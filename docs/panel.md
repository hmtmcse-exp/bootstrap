# Panels
---

### Basic Example:
<div class="panel panel-default">
    <div class="panel-heading">
        <div class="panel-title">Panel title</div>
    </div>
    <div class="panel-body">
        Panel content
    </div>
    <div class="panel-footer">Panel footer</div>
</div>


<br>
